歡迎使用《LifeTrigger 萬一我消失》（以下簡稱「本 App」或「本服務」）。我們非常重視您的個人隱私與資料安全。本隱私權政策旨在說明我們如何蒐集、使用、處理、保護及刪除您的個人資料，請您詳細閱讀。

一、我們蒐集的資料類型

為提供個人守護任務設定及定時訊息寄送服務，我們可能在您使用本 App 時蒐集以下資訊：

  • 使用者個人帳號 / 備份 Email：您設定之個人電子郵件地址（User Email）。當守護任務逾期觸發時，系統將使用此 Email 發送「安心守護 — 副本通知」給您，確保您能及時獲悉訊息發送狀況。此 Email 連同任務內文封裝於加密 Payload 中傳輸與儲存。
  • 裝置與識別碼資訊：由金流系統（RevenueCat）自動生成之高熵隨機 UUID 匿名使用者識別碼（App User ID），用於紀錄您的方案訂閱權限狀態。
  • 緊急聯絡人資料：您手動輸入之緊急聯絡人姓名（Recipient Names）、電子郵件地址（Recipient Emails）。
  • 任務與交代內容：您手動設定之守護信件訊息內文（Message）、雙方專屬之「共同記憶」（Shared Memory Prompt）及任務到期時間（Scheduled Deadline）。

二、強效金律加密處理機制（ENCRYPTION_KEY）

🛡️ 全全程端對端傳輸與強效 AES-GCM 加密

當您選擇啟用「交代守護版（保管功能）」時，您設定的使用者個人 Email、訊息內文與共同記憶在寫入伺服器（Cloudflare D1 資料庫）前，均會於伺服器端經由安全的高強度演算法（AES-GCM / 專屬 ENCRYPTION_KEY）進行加密儲存。傳輸過程全程採用 HTTPS / TLS 加密通道。

開發者無權亦無法讀取您的明文內容：即便發生資料庫被非授權存取之極端狀況，未持有安全金鑰者亦僅能看到隨機亂碼，無法還原為文字。您的個人隱私獲得最嚴格的技術保障。

三、資料保存與刪除政策

  • 地端儲存（Hive NoSQL）：免費版及安心版之任務資料僅存於您個人手機裝置中。當您在 App 內刪除任務或移除 App 時，資料將從裝置完全抹除。
  • 伺服器儲存（Cloudflare D1）：守護版資料在任務完成發送或由您主動點擊「取消任務」時，伺服器將同步標記並物理清理該筆信件內容資料，僅保留不含內容的狀態紀錄。

四、第三方服務清單與資料分享說明

我們絕不將您的個人資料出售、出租或不正當揭露給無關之第三方。本 App 僅配合以下經嚴格篩選之第三方服務供應商提供基礎技術支援：

  • Resend（郵件寄送服務）：當守護任務逾期觸發時，僅傳送發信所必須之收件人 Email 及信件內容以執行電子郵件寄送。
  • RevenueCat（App 內購與訂閱管理）：處理 Apple App Store 內購驗證與訂閱資格狀態同步。
  • Google AdMob（廣告服務）：針對免費版使用者提供非個人化廣告（Non-Personalized Ads）投放。當您升級為安心版或守護版後，廣告功能將自動停用。
  • Cloudflare（邊緣伺服器與資料庫）：提供安全的高效能 Cloudflare Workers API 運行環境與加密 D1 資料庫儲存。

五、使用者的權利與資料控制

您對自己的個人資料享有完全的掌控權：

  • 查詢與更新：您可隨時開啟 App 查閱或重新設定守護任務內容與收件人。
  • 撤回與刪除：您可隨時於 App 首頁點擊「取消任務」以刪除特定守護記錄。若欲要求刪除伺服器上記錄之所有個人資料，請直接發送電子郵件至我們的支援信箱：sampeng0206@gmail.com，我們將於 3 個工作天內處理完畢。
